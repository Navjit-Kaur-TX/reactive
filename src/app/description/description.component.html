Differences

  • Template-driven forms employ “FormsModule”
  • The template-driven approach draws logic from the template
  • Two-way data binding [(ngModel)]
  • Validation rules in the HTML template using attributes like required and pattern
  • Reactive forms are based on “ReactiveFormsModule”
  • Reactive driven forms the logic resides mainly in the component or typescript code.
  • Explicit data binding through reactive form controls
  • Programmatically using validators provided by the Angular Forms module
